==================== 第 1 题（判断题） ====================
答案：×（错误）
解析：PyAsc 不是深度学习框架，不替代 PyTorch、TensorFlow。它是用于编写昇腾 NPU 自定义算子的编程语言，
      与 Ascend C 类库接口一一对应。算子可被 PyTorch 通过 torch_npu 调用。


==================== 第 2 题（判断题） ====================
答案：√（正确）
解析：PyAsc README 与架构文档明确说明，PyAsc 编程接口与 Ascend C 类库接口一一对应，
      旨在提供与 Ascend C 接口相同的编程能力。


==================== 第 3 题（单选题） ====================
答案：D
解析：PyAsc 的五大核心模块为：编译和运行模块、Python 前端模块、AST 转 ASC-IR 模块、
      ASC-IR 定义模块、Ascend C 代码生成模块。没有 "Tensor 序列化模块"。


==================== 第 4 题（单选题） ====================
答案：B
解析：PyAsc 是一种用于编写高效自定义算子的编程语言，原生支持 Python 标准规范。
      A 错误：基于 C++ 扩展语法的是 Ascend C，不是 PyAsc。
      C 错误：PyAsc 程序需要通过编译器编译和运行时调度，最终运行在昇腾 AI 处理器上。
      D 错误：PyAsc 面向昇腾 AI 处理器，与 GPU 无关。


==================== 第 5 题（多选题） ====================
答案：A、B、C、D
解析：四项都是 PyAsc 相对 Ascend C 的特点：
      A：原生 Python 语法实现核函数开发，无需熟悉 C++ 高级模板特性。
      B：类型通过类型注解（如 asc.float32）显式指定，无需模板参数。
      C：采用 JIT 即时编译，且支持编译缓存复用，避免重复编译。
      D：直接支持 torch.Tensor 作为参数传入，无需额外的 Torch 封装层。
