==================== 第 1 题（单选题） ====================
答案：B
解析：A、C、D 都属于"使用 PyAsc 已有 API"的范畴，pip 快速安装即可。
      B 涉及修改 PyAsc 自身的接口或编译器，必须源码安装（推荐 pip install -e . 开发者模式）。


==================== 第 2 题（多选题） ====================
答案：A、B、C
解析：PyAsc 源码编译需要 Python 3.9–3.12、gcc/g++ ≥ 9.4.0（版本一致）、cmake ≥ 3.20、
      GLIBC ≥ 2.31。PyTorch 是运行 torch 输入输出 tensor 算子时才需要的可选依赖。


==================== 第 3 题（单选题） ====================
答案：B
解析：所有课程样例都通过 -r RUN_MODE 与 -v SOC_VERSION 两个参数控制运行方式。
      -r 取值为 NPU（上板）或 Model（仿真器），-v 取值为 AscendXXXXX 形式的 SOC 版本。


==================== 第 4 题（单选题） ====================
答案：A
解析：NPU 上板模式仅需 source CANN 环境变量即可执行，不需要 LD_PRELOAD 或额外的 LD_LIBRARY_PATH。
      B、C 是仿真器模式才需要的额外配置；D 错误，至少要 source set_env.sh。
