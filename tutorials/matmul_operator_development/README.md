# Matmul 算子开发教程

本教程将带你学习基于 Ascend C 的 Matmul 算子开发方法，围绕 Matmul 算子的核心概念、设计思路与开发流程展开详细讲解，涵盖 Blaze 编程范式、基础 Matmul 算子开发以及量化 Matmul 等典型场景，助力开发者掌握昇腾 NPU 上矩阵乘法算子的开发要点与实操技巧。

教程按章节划分，每个章节均包含以下内容：
- **Notebooks**：包含课程知识点与练习题，适用于自主学习或讲师引导式教学。
- **SRC**：包含课程中所有的源码，供开发者自行下载及修改。
- **Answer**：包含课后练习与实践的参考答案。

> 本教程当前仅针对 **Ascend 950 系列**产品进行验证，其它产品使用可能存在问题，欢迎开发者提出 Issue 或 PR 进行共建。

## 适用对象

- 具备 C/C++ 与 Python 基础，了解深度学习基本概念。
- 希望学习昇腾 NPU 算子开发，特别是 Matmul 相关算子的开发者。

## 学习目标

完成本教程后，你将能够：

1. 理解 Matmul 算子在 NPU 上的计算原理与数据排布方式。
2. 掌握 Blaze 编程范式与 API 的使用方法。
3. 掌握基于 Ascend C / Blaze 开发 Matmul 算子的完整流程。
4. 了解 Matmul 量化模式（如 MXFP8）的实现方法与性能优化思路。
