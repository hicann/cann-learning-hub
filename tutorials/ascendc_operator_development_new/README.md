# Ascend C 算子开发课程
本目录汇总 Ascend C 算子开发相关课程，覆盖快速入门、编程模型、算子实践、框架集成及高级特性等内容，帮助开发者系统掌握昇腾 NPU 算子开发流程与编程方法。

## 课程列表
| 课程名称 | 课程描述 | 访问链接 | 状态 |
|----------|----------|----------|------|
| 快速入门课程 | 面向初学者的前置知识与算子基础内容，帮助开发者建立 Ascend C 算子开发的基本认知 | [Quick_Start_Course](./Quick_Start_Course) | 待完善 |
| 进阶课程 | 围绕编程模型、算子实践和框架集成展开，包含 SIMD/SIMT 编程模型、C API 编程、Tensor 编程等核心内容 | [Intermediate_Course](./Intermediate_Course) | 建设中 |
| 高阶课程 | 聚焦高级特性和复杂算子实践，面向已有 Ascend C 开发基础的进阶学习场景 | [Advanced_Course](./Advanced_Course) | 待完善 |

## 课程使用说明
1. 已创建课程：点击链接可进入对应目录，查看 Notebook 教程文件或后续补充的示例代码；
2. 待完善课程：目录中通过 `.gitkeep` 保留结构，后续将逐步补充教程内容；
3. 环境要求：课程内容面向 Ascend C 算子开发场景，实际运行前请准备匹配的 CANN 开发环境。
