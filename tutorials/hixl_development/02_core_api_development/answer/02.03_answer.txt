2. √
解析：建链阶段需要建立远端访问相关的通信资源，必须在 Connect 前完成本地内存注册。

3. B
解析：aclrtMallocHost 申请 Host 内存，应使用 MEM_HOST 类型注册。aclrtMalloc 申请 Device 内存，应使用 MEM_DEVICE 类型。

4. C
解析：调用 DeregisterMem 前需先 Disconnect 断链，确保内存不再被远端使用。推荐顺序：Disconnect -> DeregisterMem -> aclrtFree。

5. A、B、C
解析：RegisterMem 需在建链前完成，建议单个 HIXL 实例注册上限 4k 个，需与 Initialize 同线程或通过 context 切换。注册后还需要 Connect 建链才能传输，不能立即传输。
