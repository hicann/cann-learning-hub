1. √。ChatDataset 会把多局较短对话装入一条固定长度 packed sequence，所以一条 batch element 与一局 Wordle 没有一一对应关系。每个 microbatch 中，每个 rank 读取 2 条，两个 rank 合计 4 条；`64 ÷ (2 × 2) = 16`，因此每个 rank 累计 16 个 microbatch、共处理 32 条 packed sequences 后，两个 rank 合计形成 global batch 64 并更新一次参数。

2. A。`chat_processor`（当前配置指向 `process_wordle_sample`）直接返回按时间排序的 role/content message 列表。ChatDataset 随后应用 chat template 并分词，再构造 shifted labels 和 assistant mask。

3. B。`warmup_steps>0` 先逐步升高学习率，`decay_type="cosine"` 在后期平滑下降，`min_lr_factor=0.1` 将终点设为初始学习率的 10%。具体比例仍需结合有效 batch、总 token 数、模型规模和验证集曲线调整。

4. A、B、C。应检查单层和中间张量能否按目标维度放置、world size 与 attention head 等维度是否整除，以及通信开销后的实测吞吐。
