01.02 课后练习答案

1. 正确。GE 是昇腾平台计算图编译和运行的控制中心，提供了图构建、图编译优化及图执行控制等功能。

2. 错误。在离线场景中，ATC 编译模型时无需昇腾设备，纯靠 Host 侧即可完成编译。

3. B。GE 的两大核心职责是图编译和图执行。

4. C。云端推理服务部署、模型结构固定的场景最适合使用离线路径（ATC + ACL）。

5. ABC。GeSession 支持实时构图编译执行（A）、支持梯度迭代（B）、单 Session 可管理多张图（C）。D 错误，GeSession 在线路径实时编译执行，无需预先生成 OM 文件。

6. B。"按执行引擎（AI Core / AI CPU 等）对算子分类，拆分形成不同子图"属于图拆分阶段。

7. ABC。图准备阶段做 inferShapeAndType/inferFormat 推导（A）；静态 shape 或动态分档场景可获得确定 shape，为后续内存规划/分配提供依据，纯动态 shape 通常需在执行期确定实际 shape/tiling（B）；也会做与硬件无关的算法层优化，如常量折叠/冗余分支消除（C）。D 错误，.om 是离线场景中由 GeModel 序列化/落盘得到的文件，且在线场景不依赖 OM 执行。

8. 错误。模型下沉的"模型加载"阶段会遍历整图算子并下发到 Device 流上，但下发后不立即执行；只有在执行阶段下发"模型执行 Task"被调度到时才触发整图执行。

9. ABCD。A 模型加载一次性下发整图但不立即执行；B 执行时只需下发一个模型执行 Task 触发整图；C 相比 Host 逐算子调度显著降低 Host/Device 交互开销；D 每次下发支持刷新 Feature Map 与输入输出内存地址。四项均正确。
