# 开发者贡献目录

本目录用于存放**社区开发者贡献**的学习材料，包括但不限于：

- 学习指导书、教程
- 示例代码与配套说明
- 专题讲解（如算子开发、性能调优等）

## 投稿说明

欢迎贡献与 Ascend/CANN 相关的学习内容。投稿前请确保：

- 内容与 CANN/Ascend 技术相关
- 结构清晰、便于读者按步骤学习
- 代码可运行、说明完整

投稿方式请参考项目根目录的贡献指南，或通过 Issue/PR 与维护者联系。

## 目录结构建议

建议按主题或作者组织子目录，例如：

```
contrib/
├── tutorials/          # 教程类
├── examples/           # 示例代码
└── guides/             # 专题指导
```

具体结构可随投稿内容灵活调整。
