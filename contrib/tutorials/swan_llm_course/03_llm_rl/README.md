# 第 03 章 · 大语言模型强化学习

## 章节定位

在第 02 章 SFT / LoRA 的基础上，引入**偏好学习与强化学习**这一阶段，覆盖从 RLHF（PPO）到 DPO、GRPO 等更现代的方案。
目标是让同学理解：为什么仅有 SFT 不够、奖励信号如何注入、训练为什么更不稳定，以及在昇腾 NPU 上跑 RL 训练时的工程权衡。
