# 第 04 章 · 大语言模型推理部署

## 章节定位

本章聚焦"训练好的模型如何对外提供服务"这一环节，覆盖在昇腾 NPU 上的推理引擎选型、权重量化、批处理与服务化部署。
预期读者在学完本章后，能在 CANNLab 上把一个微调后的 Qwen3 模型起成一个对外的 HTTP / OpenAI 兼容服务，并理解吞吐、延迟、显存之间的权衡。
